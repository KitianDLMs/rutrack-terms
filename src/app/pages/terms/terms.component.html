2. Descripción del servicio

RuTrack es una plataforma tecnológica que permite conectar usuarios con servicios de transporte y logística, facilitando la solicitud, seguimiento y gestión de viajes o entregas en tiempo real.

3. Registro de usuario

Para utilizar algunas funciones de la aplicación, el usuario deberá crear una cuenta. El registro puede realizarse mediante:

  • Correo electrónico y contraseña
  • Número de teléfono

El usuario es responsable de mantener la confidencialidad de su cuenta y de toda actividad que ocurra bajo su acceso.

4. Uso adecuado de la aplicación

El usuario se compromete a utilizar la aplicación de manera responsable y a no:

  • Intentar acceder a sistemas o datos sin autorización

5. Ubicación y seguimiento

Para ofrecer el servicio correctamente, la aplicación puede acceder a la ubicación GPS del dispositivo mientras se utiliza el servicio de transporte o seguimiento de rutas.

6. Responsabilidad
